# 🚀 Smart Event Hub

Smart Event Hub is a robust web application designed to bridge communities and event organizers. Built with the **TALL Stack** (Tailwind, Alpine, Laravel, Livewire) alternative—**Laravel, React, and Inertia.js**—it offers a Single Page Application (SPA) experience with a powerful PHP backend.

### ✨ Key Features
* **User Roles:** Admin, Event Organizer (EO), and Member (RBAC).
* **Event Discovery:** Browse upcoming events with a modern, responsive UI.
* **One-Click Registration:** Seamless event signup for members.
* **Community Management:** Empowering EOs to build and manage their own communities.

### 🛠️ Tech Stack
* **Backend:** Laravel 11 (PHP 8.2+)
* **Frontend:** React.js via Inertia.js
* **Styling:** Tailwind CSS
* **Database:** MySQL
* **Security:** Spatie Laravel Permission (RBAC)
